AnielCard.js

Parse, validate, manipulate, and display card numbers and dates in JavaScript.

Analice, valide, manipule y muestre números de tarjetas y fechas en JavaScript.

Format of use

Formato de uso

Characteristics

Caracteristicas

  • All parameters of the function anielCard must be complete.
  • Todos los parametros de la funcion anielCard deben estar completos.
  • Response value is of type Boolean .
  • El valor de respuesta es de tipo Boleano .
  • Otherwise the return value will be TRUE or FALSE
  • De lo contrario, el valor de retorno será VERDADERO o FALSO
  • If return TRUE card is VALID
  • Si retorna VERDADERO, la tarjeta es valida
  • If return FALSE card is INVALID
  • Si retorna FALSO la tarjeta es invalida
  • It is recommended that each of the inputs have a unique 'id' .
  • Se recomienda que cada uno de los inputs tenga un unico 'id'
  • For now it is only accepted that the inputs sent as arguments have been selected with 'Vanilla Js'.
  • Por ahora solo se acepta que los inputs mandados como argumentos hayan sido seleccionados con 'Vanilla Js'
// declare variables that select the inputs // declare variables que sean equivalentes al input

let name = document.getElementById('name')

let cardNumber = document.getElementById('card-number')

let cvv = document.getElementById('cvv')

let month = document.getElementById('month')

let year = document.getElementById('year')

// call the function anielCard // Llame la funcion anielCard
// all parameters refer to an input (DOM) // todos los parámetros se refieren a una entrada (DOM)

anielCard(name, cardNumber, cvv, month, year);
